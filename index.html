iFrame API service

This is a small API that takes a website url as input and checks if the website allows for iFrames or not.

Making requests

Format requests as shown below. Replace the example url, www.bbc.co.uk, with the domain to validate.

/api/url=www.bbc.co.uk

Response

The service replies with a JSON response indicating if iFraming is possible or not.

{ iframe: false }
