iFrame API service

Provide a given URL as input and the API reports back if this URL can be displayed in an iFrame or not.

{ iframe: false } indicates that it won't render in an iFrame.

Validation is both on X-Frame-Options and Content Security Policy

Requests

Format requests as shown below. Replace the example url, www.bbc.co.uk, with the domain to validate.

https://allows-iframe.netlify.app/.netlify/functions/is-iframe?url=www.bbc.co.uk

Try it

Response

The service replies with a JSON response indicating if iFraming is possible or not.

{ url: "https://www.bbc.co.uk", iframe: false }
